# Telegram 修仙自控力助手 - 命令列表

## 基础命令

**start** - 开始使用修仙助手，查看欢迎信息和功能介绍
**help** - 显示帮助信息和所有可用命令

## 任务管理命令

**task** - 创建专注任务（格式：/task 任务描述 时长）
**reserve** - 预约15分钟后开始任务（降低60%启动阻力）
**status** - 查看当前任务状态和链条信息
**stats** - 查看今日统计和历史总览

## 修仙系统命令

### 境界与修炼
**realm** - 查看当前境界、灵力和修炼状态
**mystats** - 查看个人修仙统计（渡劫、占卜、成就等）
**stones** - 查看仙石余额和盈亏统计

### 占卜天机
**divination** - 占卜天机（格式：/divination 仙石数量）
**divination_history** - 查看占卜历史记录（最近10次）
**divination_chart** - 查看占卜走势图和统计

### 境界突破
**breakthrough** - 尝试渡劫突破当前境界
**ascension** - 飞升成仙（大乘期圆满后可用）
**confirm_ascension** - 确认飞升操作

### 排行榜
**rankings** - 查看修仙排行榜（灵力榜、境界榜、飞升榜）

## 高级功能命令（计划中）

**week** - 查看本周专注统计和报告
**settings** - 个人设置（默认时长、提醒偏好等）
**export** - 导出数据（json/csv格式）
**patterns** - 查看和管理定式（下必为例原理）
**leaderboard** - 综合排行榜

## 命令使用示例

### 基础任务
```
/start
/task 学习编程 45
/reserve 准备考试 60
/status
```

### 修仙系统
```
/realm
/divination 50
/divination_history
/breakthrough
/rankings
/mystats
```

### 完整工作流
```
1. /task 学习Python 30  → 开始30分钟闭关修炼
2. [完成任务] → 自动获得30灵力和15仙石
3. /realm → 查看境界提升
4. /divination 10 → 占卜天机试试运气
5. /rankings → 看看自己在排行榜的位置
```

## 快捷回调按钮

除了命令外，bot 还提供以下快捷按钮：

- **🚀 开始修炼** - 快速创建任务
- **⚡ 查看境界** - 查看修仙状态
- **📊 修仙排行榜** - 查看排名
- **❓ 帮助** - 显示帮助信息
- **✅ 完成任务** - 标记任务完成
- **❌ 放弃任务** - 放弃当前任务

---

## 注意事项

1. **神圣座位原理**：任何任务失败都会重置所有进度
2. **灵力获取**：1分钟修炼 = 1灵力 × 境界加成
3. **仙石获取**：1分钟修炼 = 0.5仙石
4. **占卜风险**：仙石变化会影响灵力，可能导致境界升降
5. **渡劫成功率**：不同境界有不同的成功率（95%-50%）
6. **飞升条件**：大乘期 + 50000灵力
