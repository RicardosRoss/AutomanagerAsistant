### 游戏效果缩写

| 效果 | 含义 |
|------|------|
| 修为+N | cultivationXP，增加固定修为值（丹药/灵材/功法） |
| 突破+N% | breakthroughBonus，作为额外收集物时提升突破成功率 |
| 战力+N | combatPower，装备后提升战斗能力（法宝/功法） |
| 护链 | streakProtection，防止一次任务失败重置链条 |
| 时长+Nmin | durationBonus，延长专注可用时长 |
| 炼丹材料 | 可用于炼制更高阶丹药 |
| 道统决定 | 突破时决定所凝结的道基/仙基类型 |
| 治疗 | healingPower，治愈对应等级伤势（丹药/灵材） |
| 特效 | specialEffect，战斗中的特殊效果（法宝/功法） |

## 四、主修功法与修炼法门

> **品级说明**：原著功法为一到九品制（一品最次，九品最好），品级衡量**同境界内**功法的质量与神妙程度。高品级功法修炼速度远超低品级（如五品修炼速度为一品的四倍）、法力更凝实、附带更多秘法。**古法**是古代流传的原版功法（所需天地灵气可能已绝迹），品级不降；**秘法**是后人修改古法产生的弱化版，品级下降且有后遗症。
>
> **分层校订（2026-04-20）**：
> 1. 本节中的 `功法` 仅指主修功法与少量修炼法门，不再承担全部战斗技能职责。
> 2. 战斗用的 `术法 / 兵术 / 身法 / 遁法 / 辅助秘法` 另列为 `4.5 战斗法门·秘术`。
> 3. `神通` 单列为 `4.6 神通`；`求金法` 等破境法门单列为 `4.7 破境法门`。
>
> **品阶分组原则**：主修功法按品级分配到掉落品阶——凡阶(一品~二品)、黄阶(三品~四品)、玄阶(五品)、地阶(六品+/顶级古法)。品阶决定掉落概率，品级决定实际效果。当前表格仍保留旧版“游戏效果”字段作过渡说明，后续应统一改写为 `修炼倍率 / 法门承载 / 神通承载 / 道基绑定`。

### 核心规则

1. **专注即奇遇**：每次完成专注任务都有概率获得凡~地阶宝物，无需额外触发条件
2. **时长定品阶**：专注时长越长，获得高品阶宝物的概率越高
3. **收集即破关**：只有集齐对应境界的指定宝物与前置条件，才能发起突破；高阶突破不再套用统一概率模板
4. **卜卦增概率**：卜卦奇缘不直接给予宝物，仅提升下一次专注完成时获得特定类别宝物的概率
5. **类别分池**：`功法`、`战斗法门`、`神通`、`破境法门` 分池处理；其中 `神通` 只在紫府后开放，`破境法门` 以剧情/重大机缘/突破专用方式分段开放

### 品阶概率与专注时长

每次完成专注任务有概率"一无所获"，概率随时长递减。获得宝物时，各品阶按权重分配。

```typescript
// 各时长档位的完整概率表（含"一无所获"）
// getDropRates 返回的是有宝物时的品阶权重，先按 nothingRate 判定是否获得
const DURATION_TIERS = {
  // 30 分钟（标准番茄钟）：50% 一无所获
  short: {
    nothingRate: 0.50,
    weights: { 凡: 0.80, 黄: 0.12, 玄: 0.07, 地: 0.01 },
  },
  // 60 分钟（双番茄钟）：25% 一无所获
  medium: {
    nothingRate: 0.25,
    weights: { 凡: 0.40, 黄: 0.32, 玄: 0.20, 地: 0.08 },
  },
  // 75+ 分钟（深度专注）：10% 一无所获
  long: {
    nothingRate: 0.10,
    weights: { 凡: 0.35, 黄: 0.30, 玄: 0.25, 地: 0.10 },
  },
} as const;

// 实际概率 = 获得概率 × 品阶权重
// 例：30min → 获得50% × 凡80% = 40% 凡阶
```

| 30 分钟 | 50% | 40% | 6% | 3.5% | 0.5% |

| 75+ 分钟 | 10% | 31.5% | 27% | 22.5% | 9% |

| 类别 | 开放层级 | 说明 |
|------|----------|------|

| 破境法门 | 剧情 / 大境界准备 / 重大机缘 | `接引法`、`化神通秘法`、`求金法` 分阶段开放，不进入常规公共池 |

### 掉落判定流程

```typescript
function rollEncounter(durationMin: number, divinationBuff?: DivinationBuff): EncounterResult | null {
  // 1. 按时长获取概率表
  const tier = durationMin >= 75 ? 'long' : durationMin >= 60 ? 'medium' : 'short';
  const config = DURATION_TIERS[tier];

  // 2. 判定是否一无所获
  if (Math.random() < config.nothingRate) return null;

  // 3. 获得宝物 → 按品阶权重抽取
  const tierWeights = { ...config.weights };

  // 4. 应用卜卦增益（如有）
  const finalTierWeights = applyTierBuff(tierWeights, divinationBuff);

  // 5. 加权随机选品阶
  const selectedTier = weightedRandom(finalTierWeights);

  // 6. 先在该品阶内抽类别，再取具体物品
  const categoryWeights = getCategoryWeightsForTier(selectedTier);
  const finalCategoryWeights = applyCategoryBuff(categoryWeights, divinationBuff);
  const selectedCategory = weightedRandom(finalCategoryWeights);
  const item = randomItemFromPool(selectedTier, selectedCategory);

  return { item, tier: selectedTier };
}
```

### 卜卦奇缘机制

卜卦不再直接给予宝物，仅影响**下一次专注任务完成时**的掉落概率。

```typescript
interface DivinationBuff {
  categoryBoost: Partial<Record<ItemCategory, number>>;  // 特定类别掉率提升
  tierBoost: Partial<Record<Tier, number>>;              // 特定品阶掉率提升
  expiresAfterNextFocus: true;                           // 一次性的，下次专注后消耗
}

// 各卦象效果
const DIVINATION_RESULTS: Record<HexagramType, DivinationBuff> = {
  大吉: {
    categoryBoost: { 丹药: 0.15, 法宝: 0.15 },
    tierBoost: { 玄: 0.10, 地: 0.05 },
    expiresAfterNextFocus: true,
  },
  中吉: {
    categoryBoost: { 灵材: 0.10, 战斗法门: 0.10 },
    tierBoost: { 黄: 0.10 },
    expiresAfterNextFocus: true,
  },
  小吉: {
    categoryBoost: { 丹药: 0.10 },
    tierBoost: {},
    expiresAfterNextFocus: true,
  },
  凶: {
    categoryBoost: {},
    tierBoost: { 凡: 0.10 },  // 反而增加凡品概率
    expiresAfterNextFocus: true,
  },
};

// 应用卜卦增益到掉落概率
function applyTierBuff(
  rates: Record<Tier, number>,
  buff?: DivinationBuff
): Record<Tier, number> {
  if (!buff) return rates;
  const result = { ...rates };
  for (const [tier, boost] of Object.entries(buff.tierBoost)) {
    result[tier as Tier] = Math.min(result[tier as Tier] + boost, 0.95);
  }
  return result;
}

function applyCategoryBuff(
  rates: Partial<Record<ItemCategory, number>>,
  buff?: DivinationBuff
): Partial<Record<ItemCategory, number>> {
  if (!buff) return rates;
  const result = { ...rates };
  for (const [category, boost] of Object.entries(buff.categoryBoost)) {
    result[category as ItemCategory] = Math.min((result[category as ItemCategory] ?? 0) + boost, 0.95);
  }
  return result;
}
```

| 卦象 | 类别增益 | 品阶增益 | 持续 |
|------|---------|---------|------|
| 大吉 | 丹药+15%, 法宝+15% | 玄+10%, 地+5% | 下次专注 |
| 中吉 | 灵材+10%, 战斗法门+10% | 黄+10% | 下次专注 |
| 小吉 | 丹药+10% | — | 下次专注 |
| 凶 | — | 凡+10% | 下次专注 |
