## 一、提问规则

- 不再使用“尽量列出 8 个”这类上限写法，避免 NotebookLM 为了简洁主动截断条目。
- 强制加入“只收录原文明确出现名称的条目，不要根据效果概括造名”。
- 必须在问题中写清排除项，避免 `功法 / 战斗法门 / 神通 / 法器 / 丹药灵物` 混池。
- 若条目较多，强制要求末尾输出：`仍有剩余条目，可继续追问`。
- 续问模板：

```text
请继续上一轮【某一类】整理，从尚未列出的条目继续，禁止重复以下条目：……
仍只收录原文具名条目；若没有更多请直接说明已基本穷尽。
```

- 歧义校订模板：

```text
请核查以下名称在《玄鉴仙族》原文中是否确为具名神通/法门，以及其分类是否原文明确：
……
请逐项按“名称｜是否为原文具名条目｜类别是否原文明确｜若明确则类别｜备注”输出。
```

- `天下明` 更稳妥的记录法是 `昭澈心（古称天下明）`。

## 四、破境法门 / 证道法

### 4.1 凡入仙 / 开灵窍

- `接引法`｜凡人突破至胎息｜请出玄珠符种，破开仙凡之隔。

### 4.2 化神通 / 续道途

- `紫府化神通秘法`｜筑基巅峰至紫府｜包含 `阳元`、`天灯`、`宝阶`、`大璺`、`回折`、`百兵府`、`帝敕令`、`金麟`、`红蝉` 九法，用于催化仙基化神通。
- `续途妙法`｜紫府期｜通过吞食特定道基天才续接断绝道途。
- `潜蛟求变秘法`｜紫府期｜实质是坎水求道与求金的大纲法门，也可降低化第四神通难度。

### 4.3 求金 / 求闰 / 证道

- `五行求金法`｜紫府巅峰突破金丹｜以五道神通为柴火锤炼金性。
- `焜煌敛金法`｜紫府巅峰突破金丹｜可在相冲神通下强行求闰明阳。
- `三同二殊`｜求闰法｜三同两殊的五神通搭配路线。
- `四同一殊`｜求闰法｜后世容错稍高的闰法版本。
- `太阴求玄妙法`｜紫府巅峰求道｜叩问太阴果位。
- `垣下结璘真法`｜紫府突破金丹 / 结璘仙｜旁门转修法。
- `混一金丹妙法`｜金丹及以上｜求余位 / 求闰位的无上妙诀。
- `阐除阴水居法`｜紫府巅峰突破金丹｜水德借位求金。
- `阐除阳火居法`｜紫府巅峰突破金丹｜三火转化求闰。
- `阐化冲元闰法`（`闰阳法`）｜紫府至金丹｜火德衍化嬗变之术。
- `阐化讷元闰法`（`闰阴法`）｜紫府至金丹｜水德衍化嬗变之术。

### 4.4 代仙 / 旁门成道 / 更高阶路径

- `借土成道神丹心法`｜结丹 / 成神丹｜假借金位，将高位之物藏入真灵。
- `灵宝凭虚炼道正法`｜金丹 / 神丹期｜感应所寄果位并不断凝炼金性。
- `上祈灵宝玄术`｜金丹期｜感应天地果位的绝密妙诀。
- `南离服食法`｜凡人直指真君｜古仙服气养性道。
- `少明服食法`｜凡人直指真君｜通玄一脉服气成道法。

备注：
- `求金法 / 证道法` 不应与常规战斗技能池混放。
- `紫府化神通秘法` 更像突破过程资源槽或成长节点，不应当作普通掉落战技。

## 五、NotebookLM 建模锚点

### 5.1 战斗法门强度分档

- `基础 / 入门法门`｜胎息至练气前期｜效果单一，法力具象化程度低｜适合做初始技能、基础普攻、基础位移与护盾。
- `进阶 / 精妙法门`｜练气后期至筑基｜三四品正法，开始出现分化、缴械、减速、控场｜适合做中期核心技能。
- `高阶 / 紫府秘法`｜筑基巅峰至紫府｜五六品法门，带真火、异水、天象、大范围压制｜适合做后期长冷却大招。
- `准神通 / 近神通级法门`｜紫府顶级｜已接近概念压制、空间法则、替死锁血｜适合做传说级技能或 BOSS 机制。

### 5.2 神通与破境法门分档

- `明确定类神通`｜T0-T1｜高阶角色终极技能池，必须高稀有、高代价、长周期获取。
- `未明神通`｜T1-T1.5｜适合作为特定道统专属神技，增加前置条件或环境限制。
- `化神通秘法`｜T2-T1｜更适合作为突破天赋树、成长节点或进度加速器。
- `求金法 / 证道法`｜EX｜应设计为超长任务链、世界级稀缺资源或唯一位格争夺。

### 5.3 道行如何影响建模

- `战斗法门`｜道行高者学复杂术法更快，古法发挥更依赖道行｜适合把道行做成施法倍率、学习耗时减免、机制解锁门槛。
- `主修功法`｜高阶古法对道行有硬门槛，道行足够才看得懂、串得起残卷关系｜适合把道行做成功法装备门槛与隐藏羁绊解锁器。
- `神通发挥`｜入紫府后神通越来越脱离主修功法，更多取决于个人道行｜适合把道行做成神通独立天赋树和破境权重。
