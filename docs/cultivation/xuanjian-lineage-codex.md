|------|------|------|

| **五德（五行）** | 25 | 火(5+3阳)、水(5)、木(5)、金(5)、土(5含归土) |

## 三、火德 · 五火 + 三阳

### 3.1 三阳

| 道统 | 五行位 | 定位 | 核心特征 | 代表神通 | 代表人物 |
|------|--------|------|----------|----------|----------|
| **太阳** | 三阳之首·第一显位 | 刚正光明，至阳至刚 | 能五道皆明，破除魔障邪祟，封锁镇压 | 大雪绝锋、太阳衍光 | 盈昃（太阳玄君）、紫霂真人 |
| **明阳** | 三阳之一 | 煌煌帝王气象 | 讲究君臣父子、尊卑有序；极擅生发生机、强横法躯、镇压异种法力 | 谒天门、赤断镞、帝观元、君蹈危、天下明 | 魏帝李乾元、李曦明、李周巍 |
| **少阳** | 三阳之一 | 行事莫测，常带魔道色彩 | 擅长观测、穿梭与隐匿；变阳成阴 | 邪绝求、少商相火 | 少阳魔君（蒯离/蒯榭卿） |

### 3.2 明阳道统详解（核心玩家道统）

**明阳紫府神通格局 — "三术、一身、一命"（共五道）：**

| 类型 | 神通名 | 效果 |
|------|--------|------|
| 术神通×3 | **谒天门** | 显化天门与金甲仙将，隔断敌方神通运转，克制释修 |
| | **赤断镞** | 主攻伐冲杀，万乘之重与残阳杀伤，代表明阳"阴所" |
| | **帝观元** | 以君驭臣的帝王气象，能串联前几道神通（下位：长明阶） |
| 身神通×1 | **君蹈危**（上曜玄極法身） | 法躯极强，绝境中逢凶化吉，白血镇压邪祟 |
| 命神通×1 | **天下明**（昭澈心） | 主六合、衡天地，施加尊卑之心，可一言削去下属修为 |

**明阳核心功法：**

| 功法名 | 品级 | 特点 |
|--------|------|------|
| 《明华煌元经》 | 五品紫府 | 含9道秘法（阳元、天灯、宝阶、大璺、回折、百兵府、帝敕令、金麟、红蝉） |
| 《金殿煌元诀》 | 四品 | 古魏国太傅崔幕所撰 |
| 《身镇虎关宝经》 | 四品紫府 | 主修身神通『君蹈危』，倾向陷阵死战 |
| 《上府明谒经》 | 五品紫府 | 含三道秘法，成就『谒天门』 |

**明阳修炼限制：**
- 秘法互斥：第一道秘法修成后，后续秘法难度成倍增加
- 逆位命数：暗含"以子弑父"的逆位意象，极易卷入皇权因果
- 道统相斥：与晞炁水火不容，兼修可导致神通解体
- 神通顺序：『天下明』必须放在最后一道修炼

| **保木** | 收取封存 | 收取、封存、幽囚剥夺；曾主导魔道大劫，今几近断绝 | 天养瓮（蓄合、明齐、保养、泰祭） | 魔祖觜玄、魔君柘离 |

### 木德道统关系

```
正木（死而不朽）← 对立 → 角木（资生繁衍）
   ↓ 高阶谋求闰位
集木（保命恢复）

历史：保木 → 主导斩养之劫 → 被清算 → 打入魔道
      集木、更木 → 第一次仙魔大战中真君陨落 → 传承稀少
      角木、正木 → 战后繁荣
```

**金德特殊机制：**
- **四兑一庚**：纯兑金极难圆满，常以四道兑金配一道庚金调和
- **齐库抱锁**：上古大能兼并齐金与库金果位，形成几乎无解的封锁，后世极难证道
- **为兑借庚**：兑金常借用庚金法力增强杀伤

## 十、道统克制与相生关系

### 五行相克

```
火 → 克 → 金（火炼金）
金 → 克 → 木（金伐木）
木 → 克 → 土（木穿土）
土 → 克 → 水（土挡水）
水 → 克 → 火（水灭火）
```

### 关键克制对

| 克制方 | 被克方 | 说明 |
|--------|--------|------|
| 并火 | 全丹 | 并火是全丹天敌 |
| 元雷（元磁） | 全丹 | 元雷克制全丹 |
| 牡火 | 寒炁、少阴 | 牡火克制阴寒道统 |
| 合水 | 并火 | "合水厌并，并火畏合" — 大道死敌 |
| 集木、牡火 | 少阴 | 少阴独惧二者 |
| 邃炁 | 晞炁 | 邃炁化解镇压晞炁 |
| 明阳邪祟 | 魔/邪 | 明阳法力强效消磨镇压异种神通 |

### 道统相斥（不可兼修）

| 道统A | 道统B | 后果 |
|-------|-------|------|
| 明阳 | 晞炁 | 神通冲突解体，升阳损碎 |
| 正木 | 角木 | 法理对立（死而不朽 vs 资生繁衍） |
| 厥阴 | 明阳 | 互佐互伤 |

  | 'yang_surplus'  // 阳水/余位
  | 'yin'           // 阴位
  | 'deficient'     // 缺位
  | 'special';      // 特殊（三阳、炁道、并古法）

// 道统定义接口
export interface LineageDefinition {
  id: string;                    // 唯一标识符，如 'mingyang'
  name: string;                  // 中文名，如 '明阳'
  category: LineageCategory;     // 大类
  element: WuXingElement;        // 五行属性
  position?: YinYangPosition;    // 阴阳位（五行道统适用）
  description: string;           // 简短描述
  cultivationFocus: string;      // 修炼侧重点
  strengths: string[];           // 擅长领域
  weaknesses: string[];          // 弱点
  counterTo: string[];           // 克制的道统ID列表
  counteredBy: string[];         // 被克制的道统ID列表
  incompatible: string[];        // 不可兼修的道统ID列表
  famousTechniques: string[];    // 代表神通/术法
  famousFigures: string[];       // 代表人物
  rarity: 'common' | 'uncommon' | 'rare' | 'legendary' | 'extinct';
  // extinct = 传承断绝，legendary = 真君级别可见，rare = 极少修士
}
