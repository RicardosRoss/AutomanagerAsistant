当前机器人使用的是通用九境修仙模板，不等同于《玄鉴仙族》原设定。后续若要做“玄鉴化”改造，应优先以本文为设定源，再决定代码层如何实现。

## 一、修炼体系总览

### 1.1 核心结论

- `原文明确`：《玄鉴仙族》在当前语料中明确出现的基础大境界序列为：`胎息 -> 练气 -> 筑基 -> 紫府 -> 金丹 -> 元婴`。
- `原文明确`：古称体系中，`胎息` 又称 `养轮`，`练气` 又称 `服气`，`紫府` 又称 `炼神`，`金丹` 又称 `求性`。
- `原文明确`：当前语料对早期境界，尤其是 `胎息` 与 `紫府` 的描述最完整；`金丹` 与 `元婴` 已确认存在于序列中，但机制细节仍不足。
- `RAG 汇总`：《玄鉴仙族》的“修仙体系”不应只理解成单纯境界表，它至少包含三层：
  - 境界层：胎息到元婴的修为阶梯。
  - 道统层：青玄、兜玄、通玄等更高层历史道统框架。
  - 资源层：灵窍、符种、功法、灵石、灵物、法器、阵法、供奉与家族/宗门培养体系。

- `映射建议`：最优做法是采用“`六大境界 + 胎息六轮子阶段 + 其余境界阶段化展示`”的双层模型。

## 二、境界详解

### 2.1 胎息

- 标记：`原文明确`
- 定位：修行第一步，属于正式入道后的第一个大境界。
- 古称/别称：`养轮`
- 修成标志：
  - 凝聚 `玄景轮` 即算“入了胎息”。
  - 胎息的完整目标是修成 `六轮`。
- 六轮名称：
  - 第一轮：`玄景`
  - 第二轮：`承明`
  - 第三轮：`周行`
  - 第四轮：`青元`
  - 第五轮：`玉京`
  - 第六轮：`灵初`
- 胎息三关：
  - `玄景`
  - `周行`
  - `玉京`
  - 这三轮尤其困难，因为分别要在不同丹田关键位置“无中生有”凝聚灵轮。
- 核心能力变化：
  - `玄景`：入道之门，凝成后可施展基础法术。
  - `周行`：法力流转全身，可强化目力、脚力与行动能力。
  - `玉京`：诞生灵识，开始具备炼丹、炼器、布阵、使用储物袋等更高操作能力。
- 常见修行方式：
  - 通过人体之“窍”操控天地灵气，勾动太阴月华。
  - 吸纳月华、反复运转法诀、凝练月华法力。
- 风险与限制：
  - 未成玄景前，若暴露“无灵窍而修行”的事实，极易引来杀身之祸。
  - 胎息修士在资源、法器、阵法上的可支配能力很弱，严重依赖外援。
- 与下一境关系：
  - 六轮养满如月，方可进入 `练气`。
- 主要证据：
  - `0007.txt`：明确“修成胎息六轮，六轮养到如同满月后便可进入练气期”
  - `0011.txt`：明确六轮全名，并指出玄景是“入道之门”
  - `0019.txt`：明确玄景、周行、玉京为“胎息三关”
  - `0039`相关事实卡：明确玉京轮后诞生灵识，方可炼丹、炼器、布阵、使用储物袋
- 面向机器人映射建议：
  - 不要把胎息再拆成“初期/中期/后期/大圆满”这种通用模板。
  - 应直接把 `玄景/承明/周行/青元/玉京/灵初` 作为真实子阶段。

### 2.2 练气

- 标记：`原文明确 + RAG 汇总`
- 定位：胎息之后的第二大境界。
- 古称/别称：`服气`
- 修成标志：
  - 胎息六轮圆满后进入练气。
- 核心能力变化：
  - 相比胎息，练气已属于更稳定的正式修士层。
  - 在宗门、家族、坊市体系中，练气常被视为可以承担更正式职责的门槛。
- 常见修行方式：
  - 当前语料对“练气内部机制”不如胎息详细。
  - 但可以确认其已脱离“纯养轮”阶段，转向更完整的法力与道法积累。
- 风险与限制：
  - 仍然高度受限于功法、灵物、法器与宗门/家族供养。
  - 对散修而言，练气资源依旧紧缺。
- 与下一境关系：
  - 下一大境界为 `筑基`。
- 主要证据：
  - `0007.txt`：明确练气位于胎息之后、筑基之前
  - `0244.txt`：明确古称为 `服气`
  - `0238.txt`：多处通过“练气修士”“筑基修士”的对比，体现两者层级差
  - `06-fact-cards.jsonl`：`青池宗` 对治下家族所收弟子“至少能修炼到练气”
- 面向机器人映射建议：
  - 练气阶段可以保留产品层 `初期/中期/后期/圆满` 展示。
  - 但文案中应改为“练气”而非继续沿用“炼气期”通用模板。

### 2.3 筑基

- 标记：`原文明确 + RAG 汇总`
- 定位：高于练气的重要跃迁境界。
- 古称/别称：
  - 当前语料中未见稳定古称。
  - 但 `筑基` 与 `仙基` 存在强关联，语境里常以“成就仙基”指代筑基成功。
- 修成标志：
  - 修士被明确称为“筑基修士”。
  - 也会出现“成就了仙基”“铸就仙基”等表达。
- 核心能力变化：
  - 地位显著高于练气。
  - 在资源调动、交易能力、家族威望与对外话语权上发生质变。
- 常见资源：
  - 灵药、宝药、灵物支持明显更重要。
  - 普通资源对筑基以上境界帮助开始明显下降。
- 风险与限制：
  - 筑基依旧不是修行终点，上方仍有紫府压力。
  - 家族若只有一位筑基，往往仍然脆弱。
- 与下一境关系：
  - 下一大境界为 `紫府`。
- 主要证据：
  - `0238.txt`：多处把“成就仙基”与“筑基修士”并列使用
  - `0199.txt`：提到“铸就仙基”“筑基的要诀”
  - `0746.txt`：明确“成就仙基了”对应筑基层级语境
- 面向机器人映射建议：
  - 文案层建议同时保留两个词：
    - 境界名显示为 `筑基`
    - 突破成功提示可写作“铸就仙基”

### 2.4 紫府

- 标记：`原文明确`
- 定位：高于筑基的高阶境界。
- 古称/别称：
  - `炼神`
  - 常见别名：`紫府境` `紫府境界`
- 修成标志：
  - 进入紫府后，与“神通”高度绑定。
- 核心能力变化：
  - 古称“炼神”即强调这一境以神通见长。
  - 紫府修士具备强烈的局势主导能力，对家族、宗门、地域格局都有决定性影响。
- 突破机制：
  - 紫府金丹道的突破常以 `煅仙基冲上升阳` 为核心。
  - 仙基一旦飞举离开气海，几乎没有回头路，失败往往直接身死。
  - 成功后会显化神通。
- 风险与限制：
  - 紫府突破极险。
  - 一旦某条相关道路被既有紫府占住，后来者会被严重卡位。
- 额外设定：
  - 紫府修士寿命“整整五百年”。
- 与下一境关系：
  - 下一大境界为 `金丹`。
- 主要证据：
  - `0244.txt`：明确“紫府称为炼神”
  - `0726.txt`：详细描述紫府突破过程
  - `0290.txt`：明确紫府寿命整整五百年
- 面向机器人映射建议：
  - 这是最适合做“高风险突破事件”的境界。
  - 如果后续保留 `/breakthrough` 机制，应把“筑基 -> 紫府”做成最显著的一次世界观升级，而不是普通概率跳级。

### 2.5 金丹

## 三、修行要素体系

### 3.1 灵窍与玄珠符种

- `原文明确`：`灵窍是大道之基`。
- `原文明确`：正常修行对灵窍有基础要求。
- `原文明确`：法鉴通过 `玄珠符种` 能让无灵窍者也“从凡入圣”，具备接近身具灵窍者的修行可能。
- `原文明确`：符种契合度存在差异，初次修行时会显现“白毫”，白毫越长，修行速度越接近真正的灵窍者。
- `映射建议`：如果以后做《玄鉴仙族》专属 bot，最有特色的起步机制不应该是“人人都从 0 开始攒灵力”，而应该是：
  - `灵窍者`
  - `符种受授者`
  - `普通凡人`
  三类起点。

主要证据：

- `0008.txt`：白毫长度对应与玄珠符种的契合度
- `0011.txt`：明确灵窍是大道之基
- `0009.txt`：明确法鉴赐下符种，使李尺泾跨入修仙之门

### 3.2 功法

- `原文明确`：早期最核心功法是《太阴吐纳养轮经》。
- `原文明确`：该功法讲授“通过人体之窍操控天地灵气，勾动太阴月华，修成胎息六轮”。
- `RAG 汇总`：该功法在李家早期崛起中是基础修行框架。
- `原文明确`：此功法“写不出也念不出”，带有强烈保密性。
- `原文明确`：法鉴本命法决《玄珠祀灵术》负责凝聚玄珠符种，构成家族修行体系的另一条核心支柱。

主要证据：

- `0007.txt`
- `0009.txt`
- `05-entity-cards.jsonl` 中 `太阴吐纳养轮经`
- `05-entity-cards.jsonl` 中 `玄珠祀灵术`

### 3.3 灵气、月华与日常修行

- `原文明确`：修行不是抽象“经验值增长”，而是对天地灵气、太阴月华的真实牵引与炼化。
- `原文明确`：月华在早期修行中极关键。
- `原文明确`：法鉴能凝聚月华，显著提高修炼效率。
- `映射建议`：如果未来改造任务奖励，不应只用“灵力”一个抽象数值。更贴近原设定的做法是拆成：
  - `修为进度`
  - `月华/灵气积累`
  - `突破资源`

### 3.4 灵石与修行经济

- `原文明确`：灵石是修仙者世界中的硬通货，类似凡人的金银。
- `原文明确`：坊市交易、店铺结算、家族供奉都以灵石为核心计价。
- `原文明确`：早期家族对灵石极度匮乏，修行经济与凡俗农业产出高度绑定。
- `原文明确`：文中还有相对直观的兑换感：
  - 十枚白元果约可抵一枚灵石。
  - 一百斤灵稻约可抵一枚灵石。
- `映射建议`：当前 bot 的“仙石”命名不适合《玄鉴仙族》语境，应优先改为 `灵石`。

主要证据：

### 3.5 灵物、宝药与资源稀缺性

- `原文明确`：越往上修，普通灵物越不够用。
- `原文明确`：到了筑基、紫府相关层级，普通灵物效果已明显下降，更依赖“得天地精华的宝药”。
- `RAG 汇总`：《玄鉴仙族》的资源逻辑不是线性增长，而是强烈稀缺、层级分明、并且与势力结构深度绑定。
- `映射建议`：后续玩法改造时，突破不应只依赖百分比成功率，应和“灵物/宝药/供奉/机缘事件”绑定。

- `映射建议`：未来如果要扩玩法，优先扩“成品道具”和“阵法/符箓事件”，不要一上来就开放全职业百艺系统。

### 3.7 道统框架

## 四、人物与势力中的培养路径

### 4.1 家族路径

- `原文明确`：李家早期的修行崛起核心不是标准宗门入门，而是依托法鉴、玄珠符种与《太阴吐纳养轮经》。
- `原文明确`：家族修行与灵田、供奉、灵稻、白元果、护山阵法、香火祭祀紧密相连。
- `RAG 汇总`：《玄鉴仙族》的家族修仙路径非常适合改造成“经营型成长”，而不是单纯角色练级。

### 4.2 宗门路径

- `原文明确`：青池宗控制越国南五郡，并通过家族、供奉、弟子征收、丹药法器供给来统治基层。
- `原文明确`：青池宗只把控制下沉到“城”级，再借世家盘根错节地治理下面。
- `RAG 汇总`：宗门不是单纯授课机构，而是兼具统治者、供应商、收割者三重身份。

### 4.3 散修与世家路径

- `原文明确`：坊市虽然热闹，但关键传承并不自由流通。
- `原文明确`：散修能买到成品、材料、灵虫、符箓，未必买得到体系化传承。
- `RAG 汇总`：这意味着《玄鉴仙族》的修行体系天然适合“资源差距明显”的玩法设计。

## 五、面向机器人配置的映射建议

### 5.1 推荐的总体方案

推荐采用：

- `原设定层`：六大境界
- `进度展示层`：胎息六轮 + 其余境界阶段化
- `玩法层`：保留任务奖励、事件、排行榜等 bot 机制

这是比“继续沿用现有九境模板”更稳的方案，因为它同时满足：

1. 世界观更贴近《玄鉴仙族》。
2. 早期成长节奏比通用境界更细。
3. 不会因为直接砍掉阶段粒度而损失游戏性。

## 六、与当前机器人实现的差异

当前实现见：

- [docs/CULTIVATION_SYSTEM.md](./CULTIVATION_SYSTEM.md)
- [docs/cultivation-system.md](./cultivation-system.md)
- [src/config/cultivation.ts](../src/config/cultivation.ts)

主要差异如下：

1. 当前实现使用通用九境，不符合《玄鉴仙族》当前已确认六境。
2. 当前实现把所有境界都拆成相同阶段，忽略了胎息六轮的独特结构。
3. 当前实现使用 `仙石`，而《玄鉴仙族》硬通货是 `灵石`。
4. 当前实现把“飞升/大乘”作为主线终局，但当前语料中的《玄鉴仙族》主修炼框架并不是这一路。
5. 当前实现的突破更像抽象 RPG，而《玄鉴仙族》更强调资源、功法、灵物、家族与势力条件。

## 七、待确认项

以下内容目前不建议直接写死进配置：

- 金丹的稳定突破条件与阶段细则
- 元婴的能力边界与常规表现
- 练气内部更细的 canonical 划分
- 筑基是否存在稳定古称或更完整的传统叫法
- 三玄与具体玩家成长路径之间要不要发生直接映射
- 后期是否要把宗门、家族、道统作为玩法阵营系统引入

## 八、建议的后续改造顺序

如果你后面要真正把机器人改成《玄鉴仙族》风格，我建议按这个顺序做：

1. 先把展示文案从“通用修仙”改成“玄鉴口径”
2. 再把境界配置从 `9` 改成 `6 + 胎息六轮`
3. 再重做突破逻辑，把统一概率改成资源/条件驱动
4. 再决定要不要引入宗门、供奉、灵田、法器、阵法等扩展玩法

这样改动的风险最低，也最容易持续校准设定。
