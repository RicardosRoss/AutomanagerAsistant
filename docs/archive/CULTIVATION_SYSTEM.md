# 修仙游戏化系统使用文档

## 概述

这是一个基于传统修仙境界的游戏化自控力系统，将专注任务转化为修仙之旅。

## 核心功能

### 1. 境界系统（九大境界）

| 境界 | 灵力范围 | 称号 | 修炼加成 |
|------|---------|------|---------|
| 🌱 炼气期 | 0-999 | 炼气修士 | 1.0x |
| 🏔️ 筑基期 | 1000-2499 | 筑基修士 | 1.0x |
| 💊 金丹期 | 2500-4999 | 金丹真人 | 1.2x |
| 👶✨ 元婴期 | 5000-7999 | 元婴上人 | 1.3x |
| 🔮 化神期 | 8000-11999 | 化神真君 | 1.5x |
| 🌌 炼虚期 | 12000-16999 | 炼虚尊者 | 1.7x |
| ☯️ 合体期 | 17000-23999 | 合体大能 | 2.0x |
| ⚡ 渡劫期 | 24000-32999 | 渡劫期修士 | 2.5x |
| 🌟 大乘期 | 33000+ | 大乘期至尊 | 3.0x |

每个境界分为：初期、中期、后期、大圆满四个阶段

### 2. 修炼系统

**任务完成获得奖励：**
- **灵力**：1分钟 = 1灵力 × 境界加成
- **仙石**：1分钟 = 0.5仙石

**仙缘事件（随机触发）：**
- ✨ 天降灵石（5%概率）：+50仙石
- 💡 顿悟（3%概率）：本次修炼1.5倍灵力
- 💧 灵泉洗礼（2%概率）：+100灵力
- 📜 古籍秘法（1%概率）：+200灵力+100仙石

### 3. 占卜天机（八卦系统）

使用仙石占卜，根据卦象获得不同倍率：

| 卦象 | 吉凶 | 倍率 | emoji |
|------|------|------|-------|
| 坤卦 | 大凶 | -4x | ☷ |
| 艮卦 | 凶 | -2x | ☶ |
| 坎卦 | 小凶 | -1x | ☵ |
| 巽卦 | 平 | -0.5x | ☴ |
| 震卦 | 平 | +0.5x | ☳ |
| 离卦 | 小吉 | +1x | ☲ |
| 兑卦 | 吉 | +2x | ☱ |
| 乾卦 | 大吉 | +4x | ☰ |

**重要**：仙石变化会影响灵力（1仙石 = 2灵力），可能导致境界升降！

### 4. 渡劫系统

达到境界巅峰后可尝试渡劫突破：

- 每个境界有不同的成功率
- 渡劫失败会损失灵力
- 成功后进入下一境界

| 境界 | 成功率 | 失败惩罚 |
|------|--------|---------|
| 炼气期 | 95% | 0 |
| 筑基期 | 90% | -100 |
| 金丹期 | 85% | -200 |
| 元婴期 | 80% | -500 |
| 化神期 | 75% | -1000 |
| 炼虚期 | 70% | -2000 |
| 合体期 | 65% | -3000 |
| 渡劫期 | 60% | -5000 |

### 5. 飞升系统

达到大乘期且灵力≥50000时可选择飞升：

- 获得永久仙位印记 ☁️
- 保留仙石
- 灵力和境界重置
- 开启新一轮修仙之路

## 机器人命令

### 基础命令

```
/start - 踏入修仙之路
/task <描述> [时长] - 闭关修炼任务
/reserve - 预约闭关（15分钟后）
```

### 修仙系统命令

```
/realm - 查看当前境界和灵力
/breakthrough - 尝试渡劫突破境界
/ascension - 飞升（大乘期圆满后）
/confirm_ascension - 确认飞升
/rankings - 修仙排行榜
```

### 占卜系统命令

```
/divination <仙石数量> - 占卜天机
/divination_history - 查看占卜历史记录
/divination_chart - 查看占卜走势图
```

### 统计命令

```
/mystats - 查看修仙统计
/stones - 查看仙石余额和盈亏
```

## 集成到现有机器人

### 1. 在主 bot 文件中导入和注册命令

```javascript
// src/bot.js 或 src/handlers/commands.js

import CultivationCommandHandlers from './handlers/cultivationCommands.js';

// 在 bot 初始化后
const cultivationHandlers = new CultivationCommandHandlers(bot);
cultivationHandlers.registerCommands();
```

### 2. TaskService 已自动集成

TaskService 在任务完成时会自动调用 CultivationService 奖励灵力和仙石。

### 3. 修改 /start 命令展示修仙信息

```javascript
bot.onText(/\/start/, async (msg) => {
  const chatId = msg.chat.id;

  let message = `欢迎踏入修仙之路！🧙‍♂️\n\n`;
  message += `📚 基础命令：\n`;
  message += `/task <描述> [时长] - 闭关修炼\n`;
  message += `/realm - 查看境界\n`;
  message += `/divination <仙石> - 占卜天机\n`;
  message += `/rankings - 排行榜\n\n`;
  message += `完成任务获得灵力和仙石，提升境界，最终飞升成仙！`;

  bot.sendMessage(chatId, message);
});
```

## 技术架构

### 数据模型

1. **User.cultivation** - 用户修仙数据
   - spiritualPower: 灵力值
   - realm: 当前境界
   - immortalStones: 仙石
   - ascensions: 飞升次数
   - 等...

2. **DivinationHistory** - 占卜历史记录
   - 每次占卜的详细信息
   - 用于生成走势图和统计

### 核心服务

1. **CultivationService** - 修仙系统核心逻辑
   - awardCultivation() - 奖励灵力仙石
   - castDivination() - 占卜天机
   - attemptBreakthrough() - 渡劫
   - ascend() - 飞升

2. **TaskService** - 已集成修仙奖励
   - 任务完成自动调用 CultivationService

### 配置文件

- **src/config/cultivation.js** - 境界、卦象、仙缘事件配置

## 游戏性特点

1. **长期目标**：九大境界 + 飞升系统
2. **即时反馈**：任务完成立即获得灵力仙石
3. **随机性**：占卜系统 + 仙缘事件
4. **风险收益**：占卜可能让境界升降
5. **社交竞争**：排行榜系统
6. **无限可玩性**：飞升后重新开始

## 注意事项

1. **神圣座位原理保持不变**：任务失败仍会重置链条
2. **灵力独立于链条**：链条破坏不影响灵力和境界
3. **只有完成的任务才奖励**：失败任务不获得灵力仙石
4. **占卜有风险**：可能损失仙石导致灵力下降
