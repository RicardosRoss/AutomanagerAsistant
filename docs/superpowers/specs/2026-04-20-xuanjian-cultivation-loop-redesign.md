# 玄鉴修炼主循环重构设计

- 日期：2026-04-20
- 状态：已完成脑暴，待用户 review
- 作用域：`docs/cultivation/xuanjian-encounter-codex.md`、`docs/cultivation/xuanjian-combat-system.md`、`docs/cultivation/xuanjian-dev-manual.md`
- 目标：把当前“奇遇掉落即主要成长来源”的模型，重构为更贴近《玄鉴仙族》原设定的“日常修炼主循环 + 奇遇经济 + 主修功法 + 道行成长 + 战斗构筑”模型

## 一、背景

当前玄鉴改造文档里存在几类关键偏差：

1. 功法被写成了可直接提供固定修为的奇遇奖励，弱化了其“长期修炼体系”的定位。
2. 丹药、灵物、法器、功法的效果口径混杂，和战斗文档中的四维战力框架没有完全统一。
3. 奇遇系统承担了过多“主成长”职责，导致“日常修炼”“奇遇经济”“突破准备”“战斗构筑”混成一层。
4. 道行尚未被显式建模，缺少原文中“长期积累、影响术法、战斗与突破”的第二成长轴。
5. 文档缺少独立的“战斗法门/术法/神通”层，导致功法被错误地兼任了主动战斗技能系统。

## 二、原设定依据摘要

以下结论已通过 NotebookLM 对《玄鉴仙族全文》做过定向核实：

1. 功法更接近长期修炼体系，而不是一次性修为消耗品。
2. 道行适合作为长期成长资源，且同时具有全局增益与特定道统理解两层含义。
3. 大境界之间存在明显断层，同境界内高品功法、高品法器、高道行可以拉开显著差距，但通常不能轻易抹平大境界鸿沟。
4. 低阶资源、低阶法器、低阶灵物对高境界修士帮助会快速衰减。
5. 灵石适合作为前中期可获得、可消耗、可波动的长期经济资源，但高阶交易应逐渐过渡到灵物和以物易物。

说明：

- 上述 5 条属于“明确可证”的设定锚点。
- “功法倍率如何量化”“道行修正如何映射到具体百分比”属于游戏化推断，需要在实现前做一次数值回看。

## 三、设计目标

本次重构要达成的结果：

1. 把修为主来源收回到“完成专注后的基础修炼收益”。
2. 把主修功法改回“长期乘区 + 构筑核心”，不再是固定修为包。
3. 把道行加入为长期成长轴，并与主修道统理解挂钩。
4. 把灵石从“固定正增长资源”改为以奇遇事件波动为主的前中期经济资源。
5. 把丹药、灵物、法器、功法的效果语义统一到战斗文档可理解的字段体系上。
6. 把战斗法门/术法/神通显式拆成独立战斗构筑层，并与功法、道行、法器分工。
7. 在不做全系统重写的前提下，先把奇遇、战斗、突破这条主链路的文档口径统一。

## 四、设计边界

本次设计明确不做：

1. 多主修功法并行系统。
2. 全道统独立成长树。
3. 金丹以上完整经济重构。
4. 复杂坊市、拍卖、交易市场系统。
5. 全部高阶条目的逐条原文重算。

## 五、核心主循环

新的修炼主循环：

```text
完成专注
  -> 结算基础修为
  -> 应用主修功法倍率
  -> 应用道行修正
  -> 判定奇遇事件
  -> 发放/扣除灵石或宝物
  -> 如满足条件则进入对应境界破关流程
```

核心变化：

1. 日常修炼收益和奇遇收益拆开。
2. 功法从“掉落即修为”改成“装备后长期影响每次修炼收益”。
3. 奇遇更多承担经济、构筑、突破准备和特殊事件的职责。

## 六、成长结构

角色长期成长由 7 条线组成：

1. 境界/阶段
   作用：决定总需求值、可装备层级、可进入的战斗与突破内容层级。

2. 主修功法
   作用：决定修炼倍率、战力倾向、秘法承载、突破资格、道基/仙基方向。

3. 道行
   作用：决定术法运转成熟度、战斗发挥、突破稳定性与修炼效率修正。

4. 战斗法门/术法
   作用：决定日常斗法时使用的主动技能组合，包括兵术、术法、身法、遁法、辅助秘法。

5. 神通
   作用：紫府及以上的高阶主动构筑层，决定顶级斗法风格与跨层压制手段。

6. 法器/武器
   作用：决定 `攻/防/识/速` 与战斗特效。

7. 丹药/灵物
   作用：少数负责固定修为，多数负责治疗、突破、状态、构筑和特殊机制。

## 七、修为模型

### 7.1 基础修为公式

```text
基础修为 = 时长基础值 × 主修功法倍率 × 道行修正
```

解释：

1. `时长基础值`：本次专注多久，决定修炼的基础盘子。
2. `主修功法倍率`：体现功法高低品与正法/秘法差异。
3. `道行修正`：体现长期体悟对修炼效果的优化。

### 7.2 不采用“高境界吸收更少”的处理

高境界修炼更难，不通过“同样专注但吸收更少修为”来体现，而主要通过以下几项体现：

1. 当前境界总需求值更高。
2. 大境界突破条件更重。
3. 对功法品级、道行、灵物、同源外物的要求更严。

这意味着：

1. 高境界修士并不会因为境界高而天然“每次修炼都吸收更少”。
2. 高境界成长节奏主要由更大的目标盘子与更复杂的突破链路决定。

### 7.3 时长基础值建议

V1 建议值：

| 专注时长 | 时长基础值 |
|----------|------------|
| 60 - 89 分钟 | 1 |
| 90 - 119 分钟 | 2 |
| 120 - 149 分钟 | 3 |
| 150 分钟及以上 | 每多 30 分钟再 +1 |

等价公式：

```text
若 duration < 60 分钟，则不进入主修为模板；
若 duration >= 60 分钟，则
时长基础值 = 1 + floor((duration - 60) / 30)
```

说明：

1. 这是首轮建议值，不是原设定直接给出的常数。
2. 这一版模板刻意压低了基础盘子，避免功法倍率与道行修正叠乘后把节奏推得过快。
3. 后续如果要调快或调慢整体成长节奏，应优先改这里，而不是先削功法或道行。

## 八、主修功法模型

### 8.1 基本原则

玩家同时只能装备 1 部主修功法。

主修功法不再直接给固定修为，而提供长期属性：

1. `cultivationMultiplier`
2. `combatBias`
3. `artSlots`
4. `divinePowerSlots`
5. `realmCap`
6. `requiredAura`
7. `foundationBind`
8. `breakthroughAssist`

### 8.2 功法倍率建议

| 品级 | 修炼倍率 |
|------|----------|
| 一品 | 1.00 |
| 二品 | 1.10 |
| 三品 | 1.25 |
| 四品 | 1.45 |
| 五品 | 1.75 |
| 六品 | 2.10 |

修饰规则：

1. `古法`：在原品级倍率上再乘 `1.10 ~ 1.20`
2. `秘法`：在原品级倍率上再乘 `0.85 ~ 0.95`

说明：

1. 上表是“高品功法显著优于低品功法”的游戏化表达。
2. 它抓的是原文“正法珍贵、法力更凝实、秘法更多、可走更远”的精神，不是逐字逐句的原文数学公式。

### 8.3 功法的战斗职责

功法在战斗文档里应提供：

1. 战力倾向，而非单纯战力总值。
2. 战斗法门/神通承载能力。
3. 对应道统下的技能与特效发挥基底。

## 九、战斗法门与神通模型

### 9.1 分层原则

原文里的“法门”是宽口径词，在游戏化文档中必须拆开处理：

1. `主修功法`
   长期平台层，负责修炼倍率、法力质量、道统归属、道基/仙基方向。

2. `战斗法门/术法`
   主动战斗技能层，负责日常斗法手段，如兵术、剑诀、法术、身法、遁法、辅助秘法。

3. `神通`
   紫府后独立解锁的高阶技能层，按 `术神通 / 身神通 / 命神通` 区分。

4. `破境法门`
   如 `求金法`、`五行求金法`、`水火相济` 之类，只属于突破准备层，不占战斗技能槽。

### 9.2 战斗法门的构筑职责

战斗法门应作为独立主动构筑层单列，并至少覆盖以下类别：

1. `攻击术法/兵术`
   例如剑诀、戟法、火术、雷法，是主要输出手段。

2. `身法/遁法`
   负责位移、追击、脱战、抢先手。

3. `辅助秘法`
   负责勘破、护体、疗伤、破阵、临时增幅。

V1 建议槽位：

| 境界 | 法门槽位建议 |
|------|--------------|
| 胎息 | 0 ~ 1（基础小术） |
| 练气 | 2（主战 + 身法/辅助） |
| 筑基 | 3（主战 + 身法 + 辅助秘法） |
| 紫府 | 3（仍保留法门层） |
| 金丹 | 3 ~ 4 |

### 9.3 神通的构筑职责

神通不能继续并回功法说明里，而应作为紫府后的独立成长模块：

1. 紫府突破时，通常先由仙基飞举转化出第一道神通。
2. 紫府中后段与金丹准备期，继续围绕神通数量、类别与配伍构筑。
3. 金丹准备需要“五道神通”时，神通已是长期主线资源，而不是单次突破消耗品。

V1 建议槽位：

| 境界 | 神通槽位建议 |
|------|--------------|
| 胎息/练气/筑基 | 0 |
| 紫府 | 1 |
| 金丹 | 2 |

### 9.4 代表性落地原则

文档落地时至少要体现以下映射：

1. `月阙剑弧`、`大离白熙光` 等归入战斗法门/术法层。
2. `云中金落` 等归入身法/遁法层。
3. `清目灵瞳` 等归入辅助法门层。
4. `谒天门`、`君蹈危`、`天下明`、`大离书` 等归入神通层。
5. `求金法` 等继续归入破境法门层，不进入战斗技能槽。

## 十、道行模型

### 10.1 道行结构

道行采用双层结构：

1. `mainDaoLevel`
   解释：全局主道行，影响修炼、战斗、突破稳定性。

2. `schoolInsight`
   解释：主修功法/主道统对应的专精理解，影响该体系下功法、法门、神通的发挥。

### 10.2 道行获取

按用户确认的方向，采用双轨并重：

1. 连续保持专注任务不中断，稳定获得一定量道行。
2. 道斗、顿悟、破关，提供幅度不等的跃迁式提升。

### 10.3 道行修正建议

建议让道行成为温和但稳定的第二乘区：

1. `mainDaoLevel` 每个阶段提供约 `+3% ~ +6%`
2. `schoolInsight` 每个阶段提供约 `+4% ~ +8%`
3. 两者叠乘后的总上限，V1 先控制在 `1.35 ~ 1.50`

目的：

1. 让道行重要，但不盖过境界与主修功法。
2. 让“长期专注”和“道斗/顿悟/破关”都能在数值上留下稳定痕迹。

## 十一、灵石与奇遇经济

### 11.1 灵石定位

灵石不再和每次专注的基础修为绑定，而是主要来自奇遇事件波动。

前中期用途：

1. 购药
2. 买材料
3. 购买基础法器
4. 支付布阵、护道、日常损耗

高阶限制：

1. 紫府以上应逐步加入“灵物交换”“同源外物”“以物易物”的上层经济逻辑。

### 11.2 灵石事件化

奇遇事件应允许：

1. 获得灵石
2. 失去灵石
3. 交易赚差价
4. 护道开销
5. 被夺、赔付、购置、供奉

V1 建议范围：

| 境界层 | 灵石波动建议 |
|--------|--------------|
| 胎息/练气 | -30 ~ +80 |
| 筑基 | -80 ~ +200 |
| 紫府+ | 逐步降低灵石权重，转向高阶资源 |

## 十二、物品效果重映射

### 12.1 丹药

拆成 4 类：

1. 资粮型
   少数保留固定修为，如明显用于精进修为、温养法躯、积累法力的丹药。

2. 破境型
   主要映射为 `breakthroughBonus`、`stabilityBonus`、`deadgateAssist`。

3. 疗伤型
   主要映射为 `healingPower`、`injuryRecovery`、`streakProtection`。

4. 秘药型
   主要映射为特殊机制，如催化神通、洗练仙基、代价型增益、跨境界拔升。

### 12.2 灵物

拆成 4 类：

1. 资粮灵植/灵气
   少数保留固定修为。

2. 突破引材
   主要映射为 `foundationRequirement`、`breakthroughBonus`、`requiredAura`。

3. 疗伤奇珍
   主要映射为 `healingPower`、`cleanse`、`bodyRestore`。

4. 炼器炼丹材料
   主要映射为 `craftingMaterial`、`combatSpecialMaterial`。

### 12.3 法器/武器

常规法器统一只做战斗维度放大：

1. 武器类 -> 攻击主导
2. 防具/屏障类 -> 防御主导
3. 镜/符/探查类 -> 神识主导
4. 飞梭/遁器/莲台类 -> 遁速主导
5. 丹炉/辅助重器 -> 辅助型特效 + 少量四维分配

每件法器尽量保留原文神妙，并转译为 `specialEffect`。

### 12.4 高阶例外

允许少数高阶灵宝/大药保留跨系统效果，但需显式标注为“高阶例外”，不能作为全系统默认模板。

## 十三、战斗系统同步原则

`docs/cultivation/xuanjian-combat-system.md` 需要同步 3 个口径：

1. 功法从“固定修为 + 战力”改成“修炼倍率 + 战力倾向 + 法门承载/神通承载”。
2. 战斗法门/神通从功法说明中拆出，成为独立的主动构筑层。
3. 法器继续主打四维与战斗特效，不再和修炼收益混线。
4. 道行加入为长期成长层，影响：
   - 功法倍率发挥
   - 战斗法门/神通效果
   - 战斗细节判定
   - 高阶突破稳定性

## 十四、数值原则

### 14.1 同境界内可被拉开

同境界内，高品功法、高品法器、高道行应能形成明显压制。

### 14.2 跨大境界不轻易抹平

大境界差距仍是硬断层，通常不依靠装备和低阶资源抹平。

### 14.3 低阶资源快速衰减

建议增加显式衰减规则：

1. 低于当前境界 1 个大阶的资粮/法器：主效果约 `×0.35`
2. 低于当前境界 2 个大阶：主效果约 `×0.10`
3. 低于当前境界 3 个大阶及以上：主效果接近无效

说明：

1. 这条用于约束资源层级，不用于压低高境界日常修炼吸收量。

## 十七、验收标准

当以下条件满足时，可认为本轮文档设计通过：

1. 奇遇文档已明确区分“基础修炼收益”和“奇遇事件收益”。
2. 功法文档已不再把主修功法描述为固定修为包。
3. 战斗文档已把“功法 / 战斗法门 / 神通 / 法器”四层职责拆开。
4. 奇遇文档已显式存在独立的战斗法门池与神通池。
5. 道行已在奇遇文档和战斗文档中显式建模，并影响法门/神通发挥。
6. 高阶突破已不再套统一的低阶概率模板。
7. 灵石已被明确为事件波动型前中期经济资源。

## 十八、待确认项

本 spec 写定后，下一阶段仍需逐项确认：

1. 各境界总需求值是否沿用现有阈值，还是做一次整体重算。
2. `mainDaoLevel` 与 `schoolInsight` 的阶段名和阈值。
3. 哪些丹药/灵物保留固定修为，哪些必须完全转成状态/突破/治疗效果。
4. 哪些高阶灵宝属于“高阶例外层”。
5. 高阶灵石经济向以物易物过渡的具体切点。
